Để run được, ta cần phải cài thư viện SFML bằng câu lệnh:
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install libsfml-dev

Nếu câu lệnh trên không hoạt động, ta có thể cài bằng các cách khác theo link sau:
https://www.sfml-dev.org/tutorials/2.5/start-linux.php

Sau khi cài thành công thư viện SFML, ta gọi: ./chess

Nếu không được, ta có thể thử build bằng cách vào thư mục Source và gõ: make (sau khi hoàn tất cài đặt SFML)

Link demo của sản phẩm: https://youtu.be/Pev0-ZqRduw
